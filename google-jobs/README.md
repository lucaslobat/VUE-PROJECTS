# Introduction
This project is a responsive web app which consumes the API SerpAp that scrapes for Google Jobs.

# Dependencies
- Vite
- Vue Router 4
- Pinia
- Vuetify 3
- Material Design Icons
- [SerpAp] (https://serpapi.com/google-jobs-api)
- Axios
